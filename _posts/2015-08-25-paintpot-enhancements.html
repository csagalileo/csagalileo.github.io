Make the following enhancements to your PaintPot app:

  • Implement a plus button and implement a minus button, which should add/subtract 1 from the dotsize each time it is clicked. (For a hint on how to increment variables, watch this video: https://youtu.be/Odw-potCwrM)
  • Add a reset button that, when clicked, sets the dotsize back to its initial value (i.e., 5). (HINT: You'll need a second variable here to remember the initial value of the dotsize.)
  • A bug is an error or defect, that prevents the app from working the way it is supposed to. In this app, if the user continues to press ButtonMinus, the value of dotsize will eventually become negative. If dotsize is negative, what will be drawn when the user touches the screen? Try fixing this bug by using an If/Else statement. For how to conditionally set a variable, watch this video: https://youtu.be/VrzzXpJW8co (Hint: The If/Else block is found under Control in the Blocks Editor Toolbox.)
  • Create one or more of your own enhancements for your app.

If you get stuck, you can take a look at the solutions here:https://youtu.be/Pg5_khG7Zxk Complete the reflection on Google Classroom and submit your .apk file. To get the .apk file from App Inventor, package your app by selecting Build and downloading the .apk to your computer.
