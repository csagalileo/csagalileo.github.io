Here are the rules for Classcraft for you to look over: ClasscraftRules

Please complete the following two tasks today:

  • Fill out the Tech Survey. I will use this information to setup your Piazza and Classcraft accounts so do this first.
    • Tech Survey link
  • Write your first Hello World program in Java.
    1. Open Start Menu > All Programs > Programming tools > Eclipse
    2. Eclipse will ask you for a workspace. Choose a folder that you have access to (you may want to create one).
    3. Close the Welcome tab if it is displayed.
    4. Create a new Java Project by selecting File > New > Java Project
    5. Name your project "Hello" and click Finish.
    6. Create a new Java file by selecting File > New > Class
    7. Name your file "Hello". Make sure that the "public static void main(String[] args) checkbox is checked. Click Finish.
    8. Within the curly braces of the main method, add the following line:
      System.out.println("Hello world");
    9. Compile and run your program by selecting Run > Run As > Java Application
    10. Check that "Hello world" appears in the Eclipse console.
