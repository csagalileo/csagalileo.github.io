You will learn to use the Eclipse IDE today. Do the following for a Hello World program:

  1. Open Start Menu > All Programs > Programming tools > Eclipse
  2. Eclipse will ask you for a workspace. Choose a folder that you have access to (you may want to create one).
  3. Close the Welcome tab if it is displayed.
  4. Create a new Java Project by selecting File > New > Java Project
  5. Name your project "Hello" and click Finish.
  6. Create a new Java file by selecting File > New > Class
  7. Name your file "Hello". Make sure that the "public static void main(String[] args)" checkbox is checked. Click Finish.
  8. Within the curly braces of the main method, add the following line:
    System.out.println("Hello world");
  9. Compile and run your program by selecting Run > Run As > Java Application
  10. Check that "Hello world" appears in the Eclipse console.

As you should see, the Eclipse IDE does a lot of the work for you. Continue using the Eclipse IDE to try the following code segments to determine what they print out. Print statements help you debug your program by showing you the changes in your variables as they run through the program. In the below code segments, you will see what is happening in each iteration of a loop.


for(int i = 0; i<20; i++) {
  System.out.println(i);
}


for(int i = 1; i<20; i++) {
  System.out.println(i);
}


for(int i = 5; i<20; i++) {
  System.out.println(i);
}


for(int i = 0; i<20; i+=2) {
  System.out.println(i);
}


for(int i = 0; i<20; i+=3) {
  System.out.println(i);
}


for(int i = 0; i<21; i++) {
  System.out.println(i);
}


for(int i = 0; i<25; i++) {
  System.out.println(i);
}


for(int i = 20; i>0; i--) {
  System.out.println(i);
}


for(int i = 20; i>5; i--) {
  System.out.println(i);
}


for(int i = 0; i<20; i++) {
  if (i%2 == 0){
    System.out.println(i);
  }
}


for(int i = 0; i<20; i++) {
  if (i%2 == 0 && i%3 == 0){
    System.out.println(i);
  }
}


int sum = 0;
for(int i = 0; i<10; i++) {
  sum += i;
  System.out.println(sum);
}


for(int i = 0; i<20; i++) {
  for(int j = 0; i<20; i++) {
    System.out.print(i*j);
    System.out.print(" ");
  }
  System.out.println();
}
