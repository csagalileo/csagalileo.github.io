Reflection

Create a page on your portfolio named Reflection 34 under the Reflections category of your portfolio and answer the following questions:

  1. The CentralOps.net has free interactive tools that can be used to observe packet routing, latency time, and other Internet statistics. Just choose the tool you want to use from the navigation bar on the left.
    • Ping tests whether a host computer is reachable by repeatedly sending data packets from the CentralOps.net server in Dallas, TX to the host computer. For each of the following servers, write down the average round trip time (RTT).
      • google.com, yahoo.com, www.sfusd.edu, appinventor.mit.edu

      Here are some questions to answer:

      • Did any of the servers lose packets?
      • Are you able to find a server that has a really long RTT or does lose packets?
    • Traceroute lets you trace the route from the CentralOps.net server in Dallas, TX to some destination server. Here are some servers to try.
      • google.com, yahoo.com, www.sfusd.edu, appinventor.mit.edu
    • Use Use the Monitis Traceroute tool to measure the average latency between various servers on the Internet. How much time and how many hops do the following routes take:
      1. http://google.com
      2. http://whitehouse.gov
      3. http://csinquiry.org
  2. Watch this video that describes how a hacker could view your data on a public network using a packet sniffer. When and where do you use public networks, how do you use the Internet on them, and how might you protect your data?
  3. Using the tools below, try looking up the IP addresses for various websites, including their subdomains. Do you see any patterns? Now look up your school's domain name and IP address, and then the IP address for your computer or mobile device. Do you see any patterns?
    • WhoIs.com - find the IP address for a given host name
    • WhatIP.com - find the IP address of the device visiting the website
    • What's my IP? - mobile version
    • Possible searches: 1) google.com, maps.google.com; 2) www.wikipedia.org, en.wikipedia.org; 3) mit.edu, appinventor.mit.edu
