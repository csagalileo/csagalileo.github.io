Here's how to create and submit a pull request for your GitHub assignments:

  1. Before going on GitHub, make sure you have staged, committed, and pushed your changed files.
  2. On GitHub, go to the assignment repository (the original galileo2015-16 repo, not your forked repo).
  3. Click the green Compare, review, create a pull request button.
  4. On the Compare changes page, click on compare across forks.
  5. For the head fork, select your forked repo. Make sure the base fork selected is the original galileo2015-16 fork.
  6. Click the green Create pull request button.
  7. Type an appropriate title and description for your pull request that describes your changes.
  8. Click Create pull request.
