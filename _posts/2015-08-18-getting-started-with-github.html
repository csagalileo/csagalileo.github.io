  1. Create an account on Github (pick a username that resembles your real name).
  2. Follow the first part of the Hello World Github Guide to create a repository.
  3. Go to c9.io.
  4. Click on the GitHub icon in the top right to sign in using your GitHub account.
  5. Authorize Cloud 9 to link your GitHub account.
  6. You should now see your Cloud 9 Dashboard. Look for and select your repository on the left.
  7. Click on Clone to Edit.
  8. Create a Hello.java file with the code for the Hello class. 
    
public class Hello {
  public static void main (String[] args) {
    System.out.println("Hello world");
  }
}
  9. You will use javac to compile your Java files in the terminal and java to run them. Example: 
    javac Hello.java

    java Hello
  10. Use git in the terminal on the bottom of your workspace to commit and push your files to GitHub.
    Check your changed files using the git status command 
    git status

    Stage your uncommited files (this is for adding all uncommited files): 
    git add --all

    Commit your staged files: 
    git commit -m "This is your commit message."

    Push your changes to GitHub. 
    git push
