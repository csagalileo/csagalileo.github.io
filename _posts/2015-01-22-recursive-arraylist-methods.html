Some of the things you need to know how to do with ArrayList are:

  • Counting elements
  • Printing elements
  • Summing elements
  • Replacing elements
  • Finding the maximum
  • Finding the minimun
  • Adding elements
  • Removing elements

You should already be familiar with how to do all of these iteratively (using loops). In this assignment, you will write methods to do them recursively.

Fork the RecursiveArrayListMethods repository. Here is the new way we'll be developing code:

  1. Go to c9.io.
  2. Click on the GitHub icon in the top right to sign in using your GitHub account.
  3. Authorize Cloud 9 to link your GitHub account.
  4. You should now see your Cloud 9 Dashboard. Look for and select the RecursiveArrayListMethods repository on the left under Private Projects on GitHub.
  5. Click on Clone to Edit.
  6. Once Cloud 9 has finished cloning your repository, click on Start Editing.
  7. You will now see your Cloud 9 IDE with your repository files. You can click on them to open them. Remember that you should edit the .java files only, not the .class files.
  8. Complete the lab assignment.
  9. Use git in the terminal on the bottom of your workspace to commit and push your files to GitHub.
    Check your changed files using the git status command 
    git status

    Stage your uncommited files (this is for adding all uncommited files): 
    git add --all

    Commit your staged files:

    git commit -m "This is your commit message."
    

    Push(sync) your changes to GitHub.

    git push
    

  10. Create your pull request on GitHub as usual to submit the assignment.
