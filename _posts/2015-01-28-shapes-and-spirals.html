We will be drawing shapes and spirals using the pen blocks and variables.

  1. Create scripts to draw the following shapes in Scratch using the pen blocks and repeat blocks:
    • Square (when "4" key pressed)
    • Equilateral Triangle (when "3" key pressed)
    • Pentagon (when "5" key pressed)
    • Hexagon (when "6" key pressed)
    • Octagon (when "8" key pressed)
    • Circle (when "c" key pressed)
    • Five-pointed Star (when "s" key pressed)
  2. You will also create a script that will draw a variable-sided and variable-sized shape. You will need two variables for this: sides and size. Follow the format used for your other shape scripts but use the variables in place of the actual numbers. This scirpt should start when the "v" key is pressed.
  3. Lastly, you will modify the variable shapes script to draw spirals. This will require changing the size variable after each iteration in your loop. This script should start when the green flag is clicked.

Add your project to the Shapes and Spirals studio.
