Your final project will be to create a website on GitHub about a career, ethical dilemma, or community problem.

Unit 3 Final Project - Details and Rubric

You will need to create a new repository in GitHub. Choose a name for your repository that reflects what your website will be about (e.g. CareerWebsite). Use the Automatic Page Generator to create your website (or create a gh-pages branch).

You may choose to work with a partner for this project if you would like. Your project is due on Wednesday, December 17th.
