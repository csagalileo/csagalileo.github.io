The GitHub repository has been updated with a new test file for you to test if your Trainer class works. Keep in mind my method names may be different from your method names. Also note that this test file is fairly simple and does not test for every possible situation.

To get the new test file from the updated GitHub repository, you'll need to run the following commands from your terminal:

Add a remote location called "upstream":


git remote add upstream https://github.com/galileo2015-16/PokemonLabs.git

Fetch the branches from the upstream repository (for now there is only one branch called upstream/master):


git fetch upstream

Make sure you're on the master branch:


git checkout master

Merge your master branch with the upstream/master branch:


git merge upstream/master
